Descrição

Objetivo do LocPet é oferecer uma nova alternativa para que os usuários possam utilizar a ferramenta, não somente como interface para encontrar seu animal perdido, como também acessar conteúdo educativo sobre conscientização à adoção responsável. Isto feito, desejamos que seja um instrumento contributivo no despertar da população para a localização dos animais e contribua para alterar a triste e desumana prática do abandono dos animais.
